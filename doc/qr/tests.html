Overview

A set of unit tests.

Running the program

qr.test can be executed on a PDB model file but the primary use if from the clustering code. The two main functions are

  • cap the cluster with hydrogens to provide a neutral buffer layer and better SCF convergence
  • to determine the charge for each cluster
